
this reverts discord's ui font and fixes up some font weights to be close to how it looked before it was switched over to gg sans.

YOU NEED TO HAVE THE WHITNEY FONT ON YOUR MACHINE FOR THIS TO FUNCTION PROPERLY.
i got it from this site, but any of the dozens of font hosting sites will do.
https://www.cufonfonts.com/font/whitney-2

Available in two flavours:
- a BetterDiscord theme, for user convenience
- a version that can be pasted into the Custom CSS box, in case you need that.
